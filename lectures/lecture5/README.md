# 第5回課題 — プロトタイプ v1

## プロダクト概要

**論文リサーチアシスタント** — 海外論文を日本語要約 & 優先度つきで表示する Web アプリ

---

## バグ・課題

> バグリストの No.3:「研究で海外の論文を探すのが面倒」

- 英語論文を読むのに時間がかかる
- どの論文を優先して読むべきか分からない
- 論文探しで検索タブが増えていく

---

## 誰のためのプロダクト

大学・大学院で研究している人（自分自身）

---

## 動く機能（v1）

1. **arXiv 検索**: キーワードを入力するだけで最新の海外論文を取得
2. **日本語要約**: 論文アブストラクトを自動で日本語に翻訳・要約して表示
3. **優先度表示**: 関連度 + 新しさから ★★★ / ★★ / ★ で読む順番を提示

---

## 起動方法

```bash
cd prototype
pip install -r requirements.txt
python app.py
# → http://localhost:5000 を開く
```

---

## 使用技術

- **Flask** (Python) — Web サーバー
- **arXiv API** — 論文データ取得（無料・認証不要）
- **deep-translator** — 英日翻訳（Google Translate 経由）
